# 统一注释格式

### 1. 类文件注释格式

> ```
> /**
>  * File description
>  *
>  * @author ${USER}
>  * [@author ${USER}]
>  * @date ${DATE} ${TIME}
>  * @since 1.0.0
>  */
> ```



### 2. 方法注释格式

> ```
> /**
>  * Method description
>  *
>  * @author $user$
>  * [@author $user$]
>  * @date $date$ $time$
>  * [@param $param$]
>  * @return $return$
>  */
> ```
